# Initial Incident Count

I strongly recommend:

```
3 incidents
```

first.

Not 20.

Not 10.

Just 3.

Example:

```

Circular Import

fastapi-002
Route Registration Bug

fastapi-003
Dependency Injection Bug
```

If those work:

```
Create 17 more.
```
