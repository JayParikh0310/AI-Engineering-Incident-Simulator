# Column Explanation

### id

Primary key.

```
id UUIDPRIMARYKEY
```

---

### attempt_id

Links evaluation to a submission.

```
attempt_id UUID
```

FK → attempts.id

---

### model_name

Example:

```
qwen3-32b
deepseek-v3
```

Useful later when changing models.

---

### prompt_version

Example:

```
v1
v2
v3
```

Very important.

Suppose later:

```
Prompt V1
```

grades badly.

You improve it.

Now:

```
Prompt V2
```

exists.

You can compare results.

---

### raw_response

Store full LLM output.

Example:

```
{
  ...
}
```

as TEXT or JSONB.

---

### parsed_response

Store validated JSON.

Example:

```
{
  "root_cause_fixed":true,
  "confidence":0.92
}
```

Use JSONB.

---

### created_at

Timestamp.
